## Preparation

```
git clone https://github.com/mindcrunch4u/cli-search
cd cli-search

python -m venv env
source env/bin/activate

pip install -r requirements.txt
chmox +x howto
```

(Optional)
```
export HTTP_PROXY=""
export HTTPS_PROXY=""
```

## Usage

```
export OPENAI_KEY="Your OpenAI Key"
./howto download an entire directory recursively using wget?
```
